# Multivariate Exploratory Data Analysis for Agricultural Data

## Project Description
This project performs exploratory data analysis on agricultural datasets to study relationships between rainfall, yield, cost, production and profit.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Visualizations
- Histogram
- Boxplot
- Scatter plots
- Correlation heatmap

## Outcome
The analysis helps understand agricultural performance and supports data-driven farming decisions.
